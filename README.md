# Cокращатель ссылок

## Используется
- spring
- thymeleaf
- h2

## Что умееет
- некоторое api
    - показывать все данные /api/employees/
    - показывать только 1 элемент /api/employees/1
    - добавление/обновление/удаление данных

- некоторый UI
    - показывает главную со всеми данными в виде таблицы, с пагинацией
    - добавление/обновление/удаление данных
